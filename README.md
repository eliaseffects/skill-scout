# skill_scout

Live skill discovery for AI agents.

SkillScout helps you find the right skill fast. Search by task, browse trending skills, copy the install command, and you are done.

## Use it

1. Open the site.
2. Search for what you need (example: "react testing").
3. Copy the install command and run it.
